# CSL - Coop-Slot-Lobby-Documentation

"CSL - Coop Slot Lobby" is a fork from the lobby system of COALITION-Reforger-Framework. Its modified for coop PvE missions with GameMasters.

### Current Features:
- Lobby system with a main player faction and CIV faction for GM
- Slots can be configured as GM (players will get the GM rights)
- Admin-Menu with the following functions:
  - ticket system for players seeking help
  - ability to send hints to players
  - teleport system
  - healing players

### Dependencys:
- COALITION VON
